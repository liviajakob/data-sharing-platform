About

The IceExplorer is a prototype developed by Livia Jakob together with the Edinburgh Cryosphere & Earth Observation research group. The project is part of the ESA consortium CryoTop Evolution.

API

Data can be queried directly via the API. Two different endpoints are available; (1) Query available datasets and layers and (2) query data files. See the descriptions below to find out more about how to use the two endpoints.


Query available datasets:

Endpoint URL: {{root}}/v1/datasets
Example: {{root}}/v1/datasets?layertype=dem&startdate=2011-09-21

Input Parameter:

Parameter Required? Type Default Description
layertype no string Type of data. Use one of the following: dem, error, velocity, rate
startdate no string Filters only datasets with layers after this date. Format: 'YYYY-MM-DD' (e.g. 2013-11-21)
enddate no string Filters only datasets with layers before this date. Format: 'YYYY-MM-DD' (e.g. 2013-11-21)
id no integer Dataset id
protocol no string json Response type. Use one of the following: json, xml

Query data file:

Endpoint URL: {{root}}/v1/file
Example: {{root}}/v1/file?layer_id=1&date=2017-06-26

Input Parameter:

Parameter Required? Type Default Description
layer_id yes integer Layer id
date yes string Date of the layer in the following format: 'YYYY-MM-DD' (e.g. 2013-11-21)
